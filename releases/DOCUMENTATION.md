This is the first release of Hitungan (a.k.a ProCalc) developed using C++. In order to run the application, it requires the following requirements: 

(1) .NET Frameworks 4.0

(2) Visual C++ 2010 Runtime.

They can be downloaded separately from Microsoft's website. This ProCalc version contains four basic operations, i.e.: additon, subtraction, multiplication, division). The other features are modulus, square root, nth-root, nth-power, factorial, and prime number finder. For demo purpose, the application has a feature of calculation demo using 1 million big numbers. The other features are copy and paste, and file saving of the calculation result.
